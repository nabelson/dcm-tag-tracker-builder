Bulk edit · load a file to start

This tab takes a live CM360 export and lets you fix many rows at once instead of editing them one by one or wrestling with Ctrl+H across the whole sheet.

  • Filter to the rows you care about (by site, by name, or by what the target column currently contains).
  • Find & replace on one column (defaults to the click-through URL), or set the whole cell to a fixed value, scoped to just the filtered rows.
  • Export only the changed rows, with every original column and ID preserved, ready to re-import into CM360.

No file handy? Click Load sample on the left to try it on a few example rows.
